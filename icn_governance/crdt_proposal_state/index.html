Module crdt_proposal_state

Source
Expand description

CRDT-backed proposal state for conflict-free distributed governance coordination.

This module provides proposal state management using CRDTs to ensure conflict-free replication across multiple nodes. Proposals are managed using a combination of LWW-Registers for proposal metadata and CRDT Maps for votes to enable concurrent proposal management and voting without conflicts.
