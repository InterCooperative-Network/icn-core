Expand description

Defines the interface for a DAG block storage backend. Generic over the block type B which must implement Clone and be serializable/deserializable.

Required Methods§

Source

fn put(&mut self, block: &B) -> Result<(), CommonError>

Puts a block into the store. If a block with the same CID already exists, it may be overwritten or an error may be returned, depending on the implementation.

Source

fn get(&self, cid: &Cid) -> Result<Option<B>, CommonError>

Retrieves a block from the store by its CID. Returns Ok(Some(block)) if found, Ok(None) if not found, or Err on storage failure.

Source

fn delete(&mut self, cid: &Cid) -> Result<(), CommonError>

Deletes a block from the store by its CID. Returns Ok(()) if deletion was successful or if the block didn’t exist. Returns Err on storage failure.

Source

fn contains(&self, cid: &Cid) -> Result<bool, CommonError>

Checks if a block with the given CID exists in the store. Returns Ok(true) if it exists, Ok(false) if not, or Err on storage failure.
