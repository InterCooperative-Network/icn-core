CRDT-backed group membership for conflict-free distributed group management.

This module provides group membership management using CRDTs to ensure conflict-free replication across multiple nodes. Groups are managed using OR-Sets (Observed-Remove Sets) that allow concurrent additions and removals of members without conflicts.
