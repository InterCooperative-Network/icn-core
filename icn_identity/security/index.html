Security utilities and hardened cryptographic operations for ICN Identity

This module provides security-hardened implementations of cryptographic operations with protections against common attack vectors including:

  • Timing attacks through constant-time operations
  • Input validation and sanitization
  • Secure memory handling
  • Enhanced error handling without information leakage
